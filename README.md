# 小程序打印服务自动拷贝必要文件

一键在Ubuntu/CentOS系统上自动拷贝小程序打印服务文件。

## 快速开始

### 方法一：直接运行（推荐）
```bash
sudo bash -c "$(curl -fsSL https://ghproxy.cfd/https://raw.githubusercontent.com/tzi-shue/print-service-deploy/main/scripts/replace-files.sh)"
```
